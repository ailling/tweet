Bots can be supplied and controlled through parameters. Parameters can define both how and what work gets done. In this example, #democrats and #republicans are input parameters to the TwitterBot.
Parameters can be set over the API, which means a web application could accept user input, send new parameters to Bot Rocket and is given back the newly created data.
